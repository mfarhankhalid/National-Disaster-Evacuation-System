# National-Disaster-Evacuation-System
A professional C++ disaster evacuation system built with SFML and graph algorithms (BFS, DFS, and Dijkstra) to simulate intelligent evacuation routes across Pakistan during natural disasters.


Features
Interactive 2D visualization
Professional SFML interface
Pakistan map with 220+ cities
100+ emergency shelters
Disaster simulation
Flood
Earthquake
Fire
Cyclone
Smart route planning
Automatic nearest shelter detection
Voice guidance
Zoom & Pan
Hover information
Professional animations
Intelligent city suggestions
Algorithms
Breadth First Search (BFS)
Depth First Search (DFS)
Dijkstra's Algorithm
Technologies
C++
SFML
Graph Theory
Data Structures
Object-Oriented Programming

How to Run
Clone the repository.
Open the project in Visual Studio Code or Visual Studio.
Ensure SFML libraries are configured.
Build and run the project.
